gab, Get another beer, is a AUR helper inspired by the yet-another-yogurt repo.

# Warning: this is still in alpha testing, might not be the most stable.

# Dependencies.
- curl
- cjson
- gcc (for make)
- make (for make)

# Install guide
- Clone the repo: ```git clone https://github.com/Kolton-splendid/gab.git```
- Enter the directory: ```cd gab```
- Compile and install: ```makepkg -si```
  
- (or download the 'gabtwo' package from the AUR.)

# Usage
- Install package: ```gab -S <pkg>```
- Remove package: ```gab -R <pkg>```
- Search for a package: ```gab -Q <pkg>```
